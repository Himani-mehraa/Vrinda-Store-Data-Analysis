# Vrinda Store Data Analysis (Excel Dashboard Project)

## 📌 Project Overview
This project analyzes sales data of Vrinda Store using Microsoft Excel. 
The objective is to extract meaningful business insights and create an interactive dashboard.

---

## 📊 Dataset Information
The dataset contains:
- Order ID
- Date
- Customer details
- Product category
- Sales amount
- Profit
- Region

---

## 🔍 Key Analysis Performed
- Data Cleaning
- Handling Missing Values
- Pivot Table Analysis
- KPI Calculation (Total Sales, Total Orders, Profit)
- Region-wise & Category-wise Analysis

---

## 📈 Dashboard Features
- Interactive filters
- Monthly sales trend
- Top performing categories
- Sales by region
- Profit analysis

---

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

---

## 🎯 Business Insights
- Women category contributed the highest revenue (38% of total sales).
- Maharashtra region generated maximum sales among all regions.
- Sales peaked in October indicating strong festive season performance.
- Adult age group (30-49 yrs) is max contributing (~50%)
- Amazon Flipkart and Myntra are max contributing (~80%)
- Accessories category showed low profitability and needs improvement strategy.
